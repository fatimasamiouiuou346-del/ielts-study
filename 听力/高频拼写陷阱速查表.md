## ie / ei 陷阱（口诀：i before e except after c）

| 正确 | 错误 | 规则 |
|------|------|------|
| receive | recieve | after **c** → ei |
| receipt | reciept | after **c** → ei |
| ceiling | cieling | after **c** → ei |
| believe | beleive | not after c → ie |
| achieve | acheive | not after c → ie |
| field | feild | not after c → ie |
| foreign | foriegn | **例外！** |
| height | hieght | **例外！** |
| weight | wieght | **例外！** |
| science | sceince | **例外！**（其实是sc-ie-nce）|

## 词尾变化陷阱

| 正确 | 错误 | 规则 |
|------|------|------|
| truly | truely | true 去 e 加 ly |
| argument | arguement | argue 去 e 加 ment |
| judgment | judg**e**ment | 英式两种都可以，雅思一般接受 judgment |
| definitely | definatly, definately | definite + ly |
| unfortunately | unfortunatly | unfortunate + ly（保留e）|
| basically | basicly | basic + ally |
| publicly | publically | public + ly（不是 ally）|
