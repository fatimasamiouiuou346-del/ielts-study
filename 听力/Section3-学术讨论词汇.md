## 四、学生常犯的简单词汇错误

这些词"看起来都会"，但考试一紧张就写错：

| 正确拼写 | 典型错误 | 中文 |
|----------|----------|------|
| **environment** | enviro**n**ment（漏n）、enviroment | 环境 |
| **government** | gover**n**ment（漏n）、goverment | 政府 |
| **knowledge** | knowle**d**ge（漏d）| 知识 |
| **language** | lang**u**age（漏u）| 语言 |
| **argument** | argu**e**ment（多e→去e）| 争论 |
| **true → truly** | tru**e**ly（去e）| 真正地 |
| **judgment** | judge**ment**（英式也接受 judgement）| 判断 |
| **acknowledgment** | acknolw**l**edgment | 确认/致谢 |
| **therefore** | therefor（漏e）| 因此 |
| **throughout** | througout（漏h）| 贯穿 |
| **unfortunately** | unfortunatly（漏e）| 不幸地 |
| **definitely** | definatly、definately | 肯定地 |
| **separate** | se**p**erate（写成seperate→错！是a不是e）| 分开的 |
| **desperate** | desparate（a不是e！）| 绝望的 |
| **pursue** | peruse / persue | 追求 |
| **technique** | tech**n**ique（漏n）| 技术/技巧 |
| **colleague** | colleage、collegue | 同事 |
| **foreign** | for**ie**gn（ie不是ei）| 外国的 |
| **height** | h**ie**ght（ei顺序）| 高度 |
| **weight** | w**ie**ght（ei顺序）| 重量 |
| **receive** | re**cie**ve（ei → 是ei不是ie！口诀：i before e except after c）| 收到 |
| **achieve** | ach**ie**ve（ie）| 达到 |
| **believe** | bel**ie**ve（ie）| 相信 |
| **science** | sc**ie**nce（ie）| 科学 |
| **conscience** | consc**ie**nce | 良心 |
| **society** | soc**ie**ty | 社会 |

## 五、态度与观点词（高频出现在选择题/配对题）

| 听到的 | 含义 | 
|--------|------|
| It's quite challenging / demanding | 相当有挑战 |
| It was absolutely fascinating | 非常吸引人 |
| I found it really tedious / boring | 我觉得很乏味 |
| It was incredibly rewarding | 非常有收获 |
| That's absolutely ridiculous | 太荒谬了 |
| I'm not entirely convinced | 我不完全信服 |
| I totally agree / I can't agree more | 完全同意 |
| I'm not sure about that / I doubt it | 不太确定/怀疑 |
| That's a valid point | 有道理 |
| It's a complete waste of time | 完全是浪费时间 |
| I'm overwhelmed by the workload | 被工作量压垮了 |
| It's worth doing / it's not worth it | 值得做 / 不值得 |
| I wish I'd started earlier | 真希望早点开始 |
