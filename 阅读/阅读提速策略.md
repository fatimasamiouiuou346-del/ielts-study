# 阅读提速策略 — 从6.5冲7.5

**核心问题**：时间不够用  
**目标**：阅读 7.5（40题对33-34个）  
**原则**：你不是来读懂文章的，你是来找答案的。

---

## 一、60分钟怎么分——黄金时间配比

大多数考生死在 Passage 3 没时间做。标准建议"20/20/20"是错的。

| 篇章 | 时间 | 策略 |
|------|------|------|
| Passage 1 | **15-17分钟** | 最简单，必须压缩时间 |
| Passage 2 | **18-20分钟** | 中等难度，正常节奏 |
| Passage 3 | **22-25分钟** | 最难，必须有充足时间 |
| 检查 | 剩余2-3分钟 | 检查答题卡拼写 |

**你的目标**：Passage 1 省出来的3-5分钟，留给 Passage 3。

---

## 二、读文章的正确顺序（不要从头读到尾！）

### 你的旧习惯（浪费时间）
```
读完整篇文章 → 看题 → 忘光了 → 回文章重新找 → 反复回读
```

### 新方法：三遍扫描法

**第一遍：30秒速览**
- 只读标题 + 首段第一句 + 每段第一句 + 末段最后一句
- 目的：知道文章在讲什么、结构什么样
- 不答题！不细读！30秒必须完成！

**第二遍：带题扫读（核心）**
- 先看一组题的第一题（只看第一题）
- 带着关键词回文章定位
- 找到后，读那一段的上下文（3-4句足够）
- 答完第一题，再看第二题，继续往下扫
- 题目答案通常按文章顺序出现，所以你不需要跳来跳去

**第三遍：查漏补缺**
- 段落匹配题（List of Headings / Matching Information）在最后统一做
- 这时候文章你已经扫过一遍，对每段内容有印象

---

## 三、不同题型的做题顺序（重要！）

同一篇文章里的题型，**先做容易定位的，后做需要整体理解的**。

### 做题顺序

| 优先级 | 题型 | 原因 |
|--------|------|------|
| **先做** | 填空题、简答题、完成句子、图表题 | 按顺序出现，关键词明确，最容易定位 |
| **次做** | 判断题（T/F/NG，Y/N/NG） | 也基本按顺序，但需要仔细分辨 NG |
| **次做** | 选择题 | 选项干扰多，需要排除法 |
| **最后做** | 段落标题匹配（List of Headings） | 需要理解整段主旨，不能定位 |
| **最后做** | 信息匹配（Which paragraph contains...） | 乱序，需要扫全文 |

**实操**：打开一篇阅读 →
1. 30秒速览全文
2. 看有哪些题型，确定做題顺序
3. 先做填空题（有空格关键词，最快定位）
4. 再做判断题
5. 最后做匹配题

## 四、关键词定位法 —— 最核心的提速技巧

### 什么词最适合做关键词（不易被替换）

| 类型 | 例子 | 为什么好用 |
|------|------|-----------|
| 数字 | 1970, 50%, $100 | 数字不会被替换 |
| 专有名词 | London, Mars, DNA | 首字母大写，一眼看到 |
| 斜体/引号词 | *Homo sapiens* | 格式特殊，容易定位 |
| 复合名词 | climate change, heart disease | 专业性强的复合词很少被替换 |

### 什么词不适合做关键词（一定会被替换）

| 原文用词 | 题目可能替换成 |
|----------|---------------|
| important | significant, crucial, essential, vital |
| problem | issue, difficulty, challenge, obstacle |
| cause | lead to, result in, give rise to |
| increase | rise, grow, climb, soar |
| solve | address, tackle, deal with, overcome |

### 定位步骤

1. 题目中找到 **2个以上** 关键词（一个名词 + 一个限定词）
2. 扫读文章，找关键词或它的同义替换
3. 找到后，读定位句 + 前后各1句
4. 答题，然后看下一题

**黄金规则**：如果30秒扫不到关键词，跳过这题，先做下一题。不要死磕！

## 五、最难题型专项突破

### 1. TRUE/FALSE/NOT GIVEN —— 你最容易栽的坑

这题考的不是理解力，是逻辑严密性。

| 判断 | 含义 | 检验方法 |
|------|------|----------|
| **TRUE** | 题目和文章说法一致 | 能找到原文直接支持 |
| **FALSE** | 题目和文章说法矛盾 | 原文明确说了相反的话 |
| **NOT GIVEN** | 文章根本没提到 | 既找不到支持，也找不到反对 |

**NG 的三种典型情况**：
- 题目比原文**范围更窄/更具体** → NG（原文说"学生"，题目说"大学生"）
- 题目**加了比较关系**而原文没有 → NG（原文说A好B好，题目说A比B更好）
- 题目提到的东西原文**只字未提** → NG

**做题技巧**：
- TRUE 通常一眼能确认（同义替换明显）
- 在 TRUE 和 FALSE 之间纠结 → 多半是 TRUE，FALSE 需要有明确矛盾
- 在 FALSE 和 NG 之间纠结 → 问自己：原文有没有明确说"不对"？没有就是 NG
- **NG 是最常见的答案**，不要因为找不到就觉得"肯定是FALSE"

### 2. List of Headings（段落标题匹配）

**不要先读选项！** 选项太多，读了也记不住，还会干扰判断。

正确做法：
1. 读第一段 → 用自己的话总结这段在说什么（一句话）
2. 去选项里找和你的总结最接近的那个
3. 重复下一段
4. 最后剩下的选项用排除法

**判断标准**：标题 = 该段的核心观点，不是细节。如果一个选项只对应段落中的某一个例子，那就是干扰项。

### 3. Matching Information（信息匹配）

最耗时间题型，必须放到最后做。

**技巧**：
- 先读所有题目，圈出每道题的关键词
- 从 Passage 1 已经做过的定位中你大概知道哪些段落在讲什么
- 非定位不可的题，从最短的段落开始扫（信息密度低，容易判断）
- 一段可能被选多次（题目会说明 NB: You may use any letter more than once）

## 六、你的每日阅读训练方法

每天18:30-20:00的阅读时间，按以下节奏安排：

### 阶段一（Day 1-5）：分题型专项练

每天只练一种题型，不限时，把方法练熟：

| 天数 | 专项题型 | 训练重点 |
|------|---------|----------|
| Day 1 | 填空题 + 简答题 | 练关键词定位，练扫描速度 |
| Day 2 | T/F/NG 判断题 | 练逻辑分辨，NG 专项 |
| Day 3 | 选择题 | 练排除法，练干扰项识别 |
| Day 4 | List of Headings | 练段落主旨总结 |
| Day 5 | Matching Information | 练乱序定位 |

每种题型练15-20题，做完后分析每道错题是因为什么错（定位错 / 同义替换没看出来 / NG判断错 / 时间不够蒙的）。

### 阶段二（Day 6-12）：整篇限时练

每天1篇（不是3篇），限时：

| 篇章 | 限时 | 
|------|------|
| Passage 1 | 15分钟 |
| Passage 2 | 18分钟 |
| Passage 3 | 22分钟 |

每天轮换一篇。做完后不限时精读分析错题。

### 阶段三（模考日 Day 7 & 10）：全套3篇60分钟

完整模拟，重点不是分数，是**感受时间分配**。

## 七、提速口诀

1. **先看题，后读文** → 带着问题扫文章
2. **数字专名是灯塔** → 先找不会被替换的词
3. **填空判断先做，匹配最后做** → 从易到难，建立文章框架后再做难题
4. **30秒扫不到就跳** → 不死磕，不在一道题上赌气
5. **NG不是找不到，是根本没有** → 别把找不到当错误
6. **Passage 1 只给15分钟** → 简单文章省时间，留给最难的 Passage 3
7. **答案都在原文里** → 不要推理！不要脑补！不要用常识！原文写了才算

## 八、自测清单（每次做完阅读后回答）

1. Passage 1 花了多少分钟？（目标 ≤17min）
2. Passage 3 剩下多少分钟？（目标 ≥22min）
3. 有没有哪道题花了超过2分钟？
4. 错题中，多少是定位错？多少是NG判断错？多少是同义替换没看出来？
5. 做题顺序有没有按"填空→判断→选择→匹配"执行？
