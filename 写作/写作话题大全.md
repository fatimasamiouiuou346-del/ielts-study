## Task 1 · 小作文题型库

### 图表类型（必须全练）

| 类型 | 练过？ | 核心考点 |
|------|--------|---------|
| 线图 (Line Graph) | ☐ | 趋势描述（上升/下降/波动/平稳） |
| 柱图 (Bar Chart) | ☐ | 对比与排序 |
| 饼图 (Pie Chart) | ☐ | 占比描述，最大/最小/相当 |
| 表格 (Table) | ☐ | 挑选关键数据，避免罗列 |
| 双图混合 (线+柱 / 饼+表) | ☐ | 两张图之间的联系与对比 |
| 地图 (Map) | ☐ | 方位描述，过去vs现在/现在vs未来 |
| 流程图 (Process) | ☐ | 步骤顺序，被动语态 |

### Task 1 核心句型清单

**趋势描述（线图/柱图）：**
- 上升：rise / increase / grow / climb / go up / soar（暴涨）/ surge
- 下降：fall / drop / decline / decrease / go down / plummet（暴跌）/ plunge
- 波动：fluctuate / vary / oscillate
- 平稳：remain stable / level off / stabilize / stay constant
- 最高点：peak at / reach a peak of
- 最低点：hit a low of / bottom out at

**占比描述（饼图/表格）：**
- 占…比例：account for / make up / represent / constitute
- 最大：the largest proportion / the majority / the biggest share
- 最小：the smallest proportion / the minority / a tiny fraction
- 大约：approximately / around / roughly / about / just over / just under

**比较结构：**
- A is significantly higher than B
- A is roughly double/triple B
- A is slightly more/less than B
- A and B are almost equal / similar
- There is a dramatic contrast between A and B

**地图/流程图常用：**
- 被建/被拆：was built / was demolished / was replaced by
- 被扩大/缩小：was expanded / was reduced
- 在…北边/南边：to the north of / in the northern part of
- 位于：is located / is situated / lies
- 步骤顺序：first / then / next / after that / finally / subsequently
